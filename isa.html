Etiquetas de html y su funcion

1. b de bold=negrita. Indica que el texto debe ser representado con una variable bold, o negrita, de la tipografía que se esté usando. Sus etiquetas son:

2.head:para información sobre el documento

3. div> división dentro del contenido

4. a> para enlaces

5. strong> para poner el texto en negrita

6. br> para saltos de línea

7. H1>…H6> para títulos dentro del contenido

8. img> para añadir imágenes al documento

9. ol> para listas ordenadas, ul> para listas desordenadas, li> para elementos dentro de la lista

10. p> /p> Etiqueta usada para escribir párrafos de texto.

11. hr> Etiqueta utilizada para «romper» entre dos secciones de una web. Usada comúnmente como separador.

12. pre> /pre> Usada para pegar texto manteniendo el pre formato propio del texto.

13. blockquote> /blockquote> Se usan para indicar que el contenido es texto citado.

14. ol> /ol> Etiquetas para crear una lista ordenada

15.ul> /ul> Etiquetas para crear una lista des-ordenada

16. li> /li> Etiquetas que recogen el contenido de un elemento de una lista, sea ordenada o no.

17.dl> /dl> Usada para crear una lista de definiciones.

18. dt> /dt> Representa un término definido por la siguiente etiqueta dd>

19. dd> /dd> Se usa para definir los términos listados antes que él.

20.figure> /figure> Indica una figura ilustrada como parte del documento HTML5.
